Input file

1) The xlsx file (Export Processed Data option) generated by the Nanotemper Prometheus machine that has one sheet called 'Overview' with a column called 'Sample ID' (sample names), and four sheets called 'Ratio', '330nm', '350nm' and 'Scattering'. The first column of the signal sheet should be called 'Time [s]', the second column should have the temperature data and all subsequent columns the fluorescence data. In some cases, extra columns with temperature and time data are included between the fluorescence columns. These extra columns are not read by Chemelt. The order of the columns containing the fluorescence signal should match the order of the 'Sample ID' column in the 'Overview' sheet (nDSFdemoFile.xlsx).

2) The xls file generated by the ThermoFluor Assay in a qPCR machine. This file has one sheet called 'RFU' where the first row has the sample positions (header), the first column has the temperature data and all subsequent columns store the fluorescence data (qPCRdemoFile.xls).

3) The xlsx file generated by the Nanotemper Prometheus Panta instrument. This file has one sheet called ‘Overview’ with a column called 'Sample ID' with the names of the samples, and one sheet called 'Data Export' (or 'melting-scan') where all the data is stored (panta.xlsx). If present, the refolding data is not parsed.

4) The text file (.txt) generated by a QuantStudio™ 3 System instrument (quantStudio.txt).

5) The text file (.txt) generated by the Agilent's MX3005P qPCR instrument (MX3005P.txt).

6) A csv file where the first column has the temperature data and all the subsequent columns the fluorescence data. Sample names are read from the header.

7) An .xlsx file exported by the UNCLE instrument. This file contains one sheet per condition, where the sample name is located in the fifth column of the first row. The second row stores temperature data in the format 'Temp: xx, Time: xxx'. From the fourth row onward, the signal data is provided, with wavelength data in the first column and corresponding signal values in the subsequent columns.

8) A JSON file file (extension '.supr') exported by the SUPR-DSF instrument software from AppliedPhotophysics (Nicoya Lifesciences, Inc.).

9) Send us your input file and we'll add a parser.

Signal to analyse

There are several fluorescence changes that can be monitored, such as the fluorescence intensity at a given excitation and emission wavelengths, the quantum yield, the emission maximum, the fluorescence lifetime, the ratio of fluorescence intensities at two different emission wavelengths, etc (Maurice R. Eftink et al., 1994). To be able to reliably estimate thermodynamic parameters, the signal must be related to the mole fractions of the different species. The most straightforward approach is to use extensive properties, that are proportional to the amount of material in the system (Hugues Bedouelle, 2016). This way, the observed signal can be expressed as a linear combination of the signals of the different species weighted by their mole fractions:

$$ \large{ Signal(T,D) = Signal_{bkg}(T,D) + \sum_{i} f_i(T,D) \cdot Signal_i(T,D) } $$

where $Signal(T,D)$ is the observed signal, $Signal_{bkg}(T,D)$ is the background signal, $f_i(T,D)$ is the mole fraction of species $i$, and $Signal_i(T,D)$ is the signal of species $i$. $T$ and $D$ are the temperature and denaturant concentration. If the protein intrinsic fluorescence is used, the background signal can usually be neglected.

The fluorescence intensity at a given excitation and emission wavelengths is an extensive property and can be used to monitor unfolding as described above. However, the emission maximum, or the ratio signal, defined as the ratio of the fluorescence intensities at two different emission wavelengths, are not extensive properties and cannot be used directly (Hugues Bedouelle, 2016; Gabriel Žoldák, 2017).

Models

The models implemented in Chemelt are based on coupling thermal and chemical unfolding. They assume that the unfolding mechanism is the same in both conditions.


  • Reversible two-state

    • Free energy of unfolding

    We start by assuming that the protein is a monomer that (un)folds reversibly via a two-state process. The fraction of protein in the native ($f_n$) and denatured ($f_u$) states depend on the unfolding equilibrium constant $K$.

    $$ \large{ f_n(T,D) = \frac{1}{1 + K(T,D)} } $$ $$ \large{ f_u(T,D) = 1 - f_n(T,D) } $$

    where $T$ is the temperature in Kelvin units and D is the denaturant concentration in molar units. The unfolding equilibrium constant $K$ is related to the Gibbs free energy $\Delta G$ by:

    $$ \large{ K(T,D) = \exp\!\left( -\frac{\Delta G(T,D)}{R\,T} \right) } $$

    where $R$ is the gas constant. If only chemical denaturation is considered, $\Delta G$ can be estimated using the Linear Extrapolation Model (LEM):

    $$ \large{ \Delta G(D) = \Delta G_{H_2O} - m \cdot D } $$

    where $m$ is an empirical parameter that allows modelling $\Delta G$ with a linear dependence on denaturant concentration. $\Delta G_{H_2O}$ is the free energy of unfolding in the absence of a denaturant agent. On the other hand, if thermal denaturation is considered, the free energy of unfolding can be calculated as indicated below:

    $$ \large{ \Delta G(T) = \Delta H_m \cdot (1 - \frac{T}{T_m}) + \Delta C_{p} \cdot (T - T_m - T \cdot \ln(\frac{T}{T_m})) } $$

    where $\Delta H$ is the enthalpy of unfolding, $T_m$ is the temperature of melting, and $\Delta C_p$ is the heat capacity of unfolding. The combination of both denaturation methods results in the expression:

    $$ \large{ \Delta G(T,D) = \Delta H_m \cdot (1 - \frac{T}{T_m}) + \Delta C_{p} \cdot (T - T_m - T \cdot \ln(\frac{T}{T_m})) - m \cdot D } $$

    Equation of the signal

    The total signal is assumed to be a linear combination of the signal produced by the different protein states weighted by their mole fractions.

    $$ \large{ Signal(T,D) = S_n(T,D) \cdot f_n(T,D) + S_u(T,D) \cdot f_u(T,D) } $$

    where $S_n(T,D)$ and $S_u(T,D)$ describe the signal dependence on temperature $T$ (in Kelvin units) and denaturant concentration $D$ (in molar units) for the native and unfolded states, respectively. The background signal is assumed to be negligible. The temperature dependence of the signal can be modelled using a constant, linear, quadratic or exponential function. The baselines parameters can be either local (different for each curve) or shared (same for all curves). The following alternatives are implemented:

    Local baselines and local slopes

    In this model, each unfolding curve with index $i$ has its own set of baseline parameters.

    $$ \large{ Signal_{x,i}(T,D) = f_{x}(T,D) \cdot (b_{x,i} + k_{x,i} \cdot dT + q_{x,i} \cdot dT^2) (\text{quadratic}) } $$ $$ \large{ Signal_{x,i}(T,D) = f_{x}(T,D) \cdot (\gamma_{x,i} + \delta_{x,i}\, e^{-\alpha_{x,i}\,\Delta T}) (\text{exponential}) } $$

    where $f_{x}(T,D)$ is the fraction of native ($x=n$) or unfolded ($x=u$) protein at temperature $T$ and denaturant concentration $D$, and $dT$ is the difference between the temperature $T$ and a reference temperature ($298K$).

    Local baselines and shared slopes

    In this model, the slope terms ($k_{x}$ and $q_{x}$ or $\delta_{x}$ and $\alpha_{x}$) are shared among all curves, but the intercept terms are local to each curve.

    $$ \large{ Signal_{x,i}(T,D) = f_{x}(T,D) \cdot (b_{x,i} + k_{x} \cdot dT + q_{x} \cdot dT^2) (\text{quadratic}) } $$ $$ \large{ Signal_{x,i}(T,D) = f_{x}(T,D) \cdot (\gamma_{x,i} + \delta_{x}\, e^{-\alpha_{x}\,\Delta T}) (\text{exponential}) } $$

    Shared baselines and shared slopes

    In this model, a linear dependence on denaturant concentration is included in the baseline equations. All baseline parameters are shared among all curves.

    $$ \large{ Signal_{x}(T,D) = f_{x}(T,D) \cdot (a_{x} + b_{x} \cdot dT + c_{x} \cdot D + d_{x} \cdot dT^2) (\text{quadratic}) } $$ $$ \large{ Signal_{x}(T,D) = f_{x}(T,D) \cdot (\gamma_{x} + \delta_{x}\, e^{-\alpha_{x}\,\Delta T} + c_{x} \cdot D) (\text{exponential}) } $$

    Shared baselines and shared slopes with scale factor

    To account for protein concentration errors or differences in detection sensitivity across positions, a scaling factor $s_i$ is included for each curve $i$.

    $$ \large{ Signal_{x,i}(T,D) = s_i \cdot f_{x}(T,D) \cdot (a_{x} + b_{x} \cdot dT + c_{x} \cdot D + d_{x} \cdot dT^2) (\text{quadratic}) } $$ $$ \large{ Signal_{x,i}(T,D) = s_i \cdot f_{x}(T,D) \cdot (\gamma_{x} + \delta_{x}\, e^{-\alpha_{x}\,\Delta T} + c_{x} \cdot D) (\text{exponential}) } $$
Choosing a model

There are several options to fit the unfolding data, depending on the number of parameters for the baselines and if the baseline parameters are shared or local. Ideally, one should choose the model with the least number of parameters that is still able to explain the observed data. For example, for ten denaturant concentrations these are the number of parameters:

Therefore, fittings with global intercepts and slopes, and no scaling factors are preferred. If that model fails to fit the data, one should try first using the scaling factor, that takes into account protein concentration errors or differences in detection sensitivity across positions. Another recommendation to reduce the number of parameters is to restrict the temperature range of analysis such that the baseline terms can be well approximated by a linear (or constant) function.

Furthermore, fitting thermochemical denaturation data is an ill-posed problem, and different sets of parameters can fit the data. If the obtained parameters do not make biophysical sense, such as a heat capacity of unfolding of 0.1 kcal/K/mol, for a protein of 130 residues, then another model should be used.
